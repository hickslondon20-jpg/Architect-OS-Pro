## Backend — `routers/domain_agents.py` (founder-auth `Depends(get_current_user_id)`)
- `GET /api/domain-agents` — list 5 agents mapped to the frontend `DomainAgent` shape (see
  `04-RESEARCH.md` field map); include `activity` (task counts / last run) + derived `initial`.
- `GET /api/domain-agents/{key}` — profile: `capabilities`, `thought_starters`, **real** workflows
  (join `workflows`), recent activity (this agent's recent tasks/artifacts).
- `GET /api/artifacts` — Library list for the founder (filter by agent/type/date/workflow);
  reuse `ArtifactService`; each row carries provenance → task/workflow/agent.
- `POST /api/domain-agents/{key}/freeform` — insert `freeform_requests` (raw text, agent,
  timestamp); attempt mapping to a workflow (reuse the VCSO `classify()`/keyword matcher); if
  mappable, `create_task(origin='profile')` and return the task; else return `mapped=false` (net-new
  assembly deferred — L16, later slice).

## Backend — `routers/tasks.py`
- `GET /api/tasks` — Kanban list for the founder; filters `agent`, `status`, `date`; response shape
  matches the frontend `DomainTask`/`types.ts`. (Task **execution** endpoints already exist — Phase 2.)

## Frontend wiring (replace mock reads; keep layout/tokens)
- New `lib/domainAgentsApi.ts` client (fetch + types) for the endpoints above.
- **Gallery** — 5 agents from `GET /api/domain-agents` + recent tasks strip from `GET /api/tasks`.
- **Profile** — `GET /api/domain-agents/{key}`; render capabilities, thought-starters, the **real**
  workflows shelf (P&L only for Financial today — do not render mock extras), recent activity, and
  the free-form input (posts to `/freeform`).
- **Kanban (Tasks)** — `GET /api/tasks`; columns Ready→Running→Blocked→Review→Done; "waiting on you"
  flag on Blocked; card → Workspace (04-02).
- **Artifacts Library** — `GET /api/artifacts` list + `GET /api/artifacts/{id}` preview + Download;
  provenance link back to task→workflow→agent; Second Brain button present but **inert** (Phase 5).
- Keep `mockDomainAgents.ts` types (`types.ts`) as the shared shape; delete/replace only the mock
  data source.

## Acceptance criteria

   agents + P&L workflow; Kanban states map to the Phase‑2 machine.
2. A free-form ask writes a `freeform_requests` row; a mappable ask launches a task; an unmappable
   one returns `mapped=false` (no crash, no fabricated workflow).
3. Skills/Templates never render; AI Usage links to Settings; Second Brain button inert.
4. `python -m compileall python-backend` + `npm run build`/focused TS check pass; live-smoke gaps
   flagged.
5. Trackers + `04-COMPLETION.md` updated (or a combined Phase-4 completion after 04-02).
