## Schema

### Per-user tables (RLS by `user_id`)

**`wiki_pages`** — one row per fixed page per user (the 7 from L8).
```sql
CREATE TABLE public.wiki_pages (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id       UUID NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE,
  page_key      TEXT NOT NULL,                 -- e.g. 'business_context' (enum in 03-02)
  title         TEXT NOT NULL,
  one_line      TEXT,                          -- A4 index entry; feeds digest
  page_kind     TEXT NOT NULL,                 -- 'compiled_base_only' | 'insight_accreting'
  wiki_version  TEXT NOT NULL,                 -- contract/schema-object version
  last_compiled_at TIMESTAMPTZ,
  stale         BOOLEAN NOT NULL DEFAULT false,
  embedding     VECTOR(1536),                  -- L4: all pages embedded (text-embedding-3-small)
  created_at    TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  updated_at    TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  UNIQUE (user_id, page_key)
);
```

**`wiki_claims`** — the unit of knowledge (L5). `class` carries the three content classes (L2).
```sql
CREATE TABLE public.wiki_claims (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id       UUID NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE,
  page_id       UUID NOT NULL REFERENCES public.wiki_pages(id) ON DELETE CASCADE,
  page_key      TEXT NOT NULL,
  text          TEXT NOT NULL,
  class         TEXT NOT NULL,                 -- 'compiled' | 'insight' | 'override'
  status        TEXT NOT NULL,                 -- 'active' | 'quarantined' | 'trusted' | 'retired' | 'contested'
  confidence    TEXT NOT NULL DEFAULT 'medium',-- L3 display enum: 'high'|'medium'|'low'
  recall_score  REAL NOT NULL DEFAULT 0,       -- L3 hidden machine score (B3 promotion gate)
  embedding     VECTOR(1536),                  -- L4: claims embedded for wiki_search
  superseded_by UUID REFERENCES public.wiki_claims(id),  -- override suppression (precedence)
  created_at    TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  updated_at    TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
-- CHECK enforced via app + trigger: class='compiled' rows may only be written by the
-- compilation service role path (sub-phase 04). No agent/founder write path inserts class='compiled'.
```

**`wiki_evidence`** — weighted, line-level provenance (L5; B1/A2).
```sql
CREATE TABLE public.wiki_evidence (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id       UUID NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE,
  claim_id      UUID NOT NULL REFERENCES public.wiki_claims(id) ON DELETE CASCADE,
  source_id     TEXT NOT NULL,                 -- doc id / supabase record id
  source_kind   TEXT NOT NULL,                 -- 'raw_document'|'document_chunk'|'tier0_record'|'global_checkpoint'
  path          TEXT,                          -- folder/doc path or table.column
  lines         INT4RANGE,                     -- line range within source
  weight        REAL NOT NULL DEFAULT 1.0,
  note          TEXT
);
```

**`wiki_contradictions`** — claim-vs-claim/page contradiction records (A3/B8).
```sql
CREATE TABLE public.wiki_contradictions (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id       UUID NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE,
  claim_id      UUID NOT NULL REFERENCES public.wiki_claims(id) ON DELETE CASCADE,
  against_claim_id UUID REFERENCES public.wiki_claims(id),
  against_page_ref TEXT,
  note          TEXT,
  resolved      BOOLEAN NOT NULL DEFAULT false,
  created_at    TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```

**`wiki_insight_records`** — write-back / dreaming candidates with trust + action-boundary metadata.
```sql
CREATE TABLE public.wiki_insight_records (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id       UUID NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE,
  claim_id      UUID REFERENCES public.wiki_claims(id) ON DELETE CASCADE, -- the insight-class claim
  trust_state   TEXT NOT NULL DEFAULT 'quarantined', -- 'quarantined'|'promotion_candidate'|'trusted'|'rejected'
  origin        TEXT NOT NULL,                 -- 'domain_agent_writeback'|'dreaming'
  novelty_ok    BOOLEAN,                       -- D11 gate ii
  about_business_ok BOOLEAN,                   -- D11 gate i
  confidence_bar_ok BOOLEAN,                   -- D11 gate iii (bar=medium, L9)
  recall_count  INT NOT NULL DEFAULT 0,        -- B3 recall frequency
  query_diversity REAL NOT NULL DEFAULT 0,     -- B3 query diversity
  -- action-boundary metadata (OpenClaw action-sensitive memory)
  safe_to_act_after TIMESTAMPTZ,
  expires_at    TIMESTAMPTZ,
  authority_owner TEXT,
  avoid_note    TEXT,
  created_at    TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  updated_at    TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```

**`wiki_action_log`** — append-only audit trail (A4) enabling reversible promotion (B4).
```sql
CREATE TABLE public.wiki_action_log (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id       UUID NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE,
  action        TEXT NOT NULL,                 -- 'compile'|'propose_insight'|'set_confidence'|
                                               -- 'flag_contradiction'|'add_override'|'promote'|
                                               -- 'demote'|'retire'|'consolidate'
  actor         TEXT NOT NULL,                 -- 'compilation_service'|'domain_agent'|'founder'|'dreaming'
  page_key      TEXT,
  claim_id      UUID,
  payload       JSONB,                         -- before/after for reversibility
  created_at    TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```

**`wiki_digest`** — one compiled digest per user (B2).
```sql
CREATE TABLE public.wiki_digest (
  user_id       UUID PRIMARY KEY REFERENCES auth.users(id) ON DELETE CASCADE,
  wiki_version  TEXT NOT NULL,
  generated_at  TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  digest        JSONB NOT NULL                 -- the §4.2 object
);
```

### Global tables (NO RLS for founders — service-role only; L6)

**`global_ip_pages`** — authored narrative IP, tagged for stage-primer targeting (§5 spec).
```sql
CREATE TABLE public.global_ip_pages (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  slug          TEXT NOT NULL UNIQUE,
  title         TEXT NOT NULL,
  body          TEXT NOT NULL,
  domain        TEXT,                          -- financial|team|clients|...
  ladder_stage  INT,                           -- AE Ladder 1..5 (Rising/Striving/Thriving/Driving/Arriving); model all 5, content currently 1-4, stage-5 = known gap (CONTEXT §8 2026-06-30)
  revenue_tier  TEXT,
  topic         TEXT[],
  version       TEXT NOT NULL,
  embedding     VECTOR(1536),
  updated_at    TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
-- The checkpoint family is NOT recreated here. Per 01-01-DELTA §A the real GM tables are
-- gm_checkpoints (125) + gm_audit_questions / gm_checkpoint_stage_meaning / gm_checkpoint_scoring
-- (500 rows each = 125 x 4 stages). Reach them via global_ip_get / structured_data using the §A
-- join (q.stage_id = ae_frontend_stage_id <-> gm_stages.stage_id). source_kind = 'global_checkpoint'.
```

### RLS
- All `wiki_*` tables: `USING (user_id = auth.uid())` SELECT/INSERT/UPDATE/DELETE policies,
  mirroring `kb_folders`.
- `global_ip_pages`: **no founder-facing policy.** Reachable only by service-role clients
  (the `global_ip_get` path and the stage-primer). A founder JWT must not be able to read it.
- Compiled-base write-lock: a `BEFORE INSERT/UPDATE` trigger on `wiki_claims` rejects
  `class='compiled'` writes unless the session carries the compilation-service marker
  (service-role + a guarded RPC). Document the exact mechanism chosen in the migration.

### Indexes
- `wiki_claims (user_id, page_key, class, status)`; ivfflat/hnsw on `wiki_claims.embedding`
  and `wiki_pages.embedding`; `wiki_action_log (user_id, created_at)`; `global_ip_pages`
  on `(domain, ladder_stage, revenue_tier)` and hnsw on `embedding`.

## Success criteria
1. Migration applies cleanly; all tables + indexes + RLS exist.
2. Founder JWT can read its own `wiki_*` rows and **cannot** read `global_ip_pages`.
3. Compiled-base write-lock trigger rejects a non-service `class='compiled'` insert.
4. `pgvector` columns present on pages, claims, and global IP pages.
5. Shapes match `../02-interface-contract/02-01-CONTRACT.md` `claim{}` / `evidence{}` exactly.
6. `generate_typescript_types` re-run; new tables typed for the frontend.

## Out of scope
Compilation logic, embeddings population, handler logic, UI. Schema + RLS + triggers only.
